# canDynamix
자율주행 자동차 프로젝트 입니다. (ROS기반)


1. 라즈베리파이  : 메카솔루션
2. DC 6V 90RPM n20 (encoder) : 알리익스프레스

3. n20용 브라켓 : 메카솔루션
4. n20용 wheel : 메카솔루션
5. [Pololu] 1/2" 메탈 볼캐스터 (Pololu Ball Caster with 1/2″ Metal Ball) : 메카솔루션
6. 서포트 (35mm) :  디바이스마트(금속), 메카솔루션(플라스틱)
7. 9V 배터리 홀더 : 메카솔루션
8. 9V 충전식 배터리 : 알리익스프레스
